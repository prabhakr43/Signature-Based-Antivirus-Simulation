Signature-Based Antivirus Simulation This project is a Python-based security tool developed during my internship at SkillFied Mentor. It demonstrates how modern antivirus software identifies and isolates threats using digital fingerprints.
Key Features
* MD5 Hashing: Generates unique digital fingerprints for every file.
* Threat Detection: Compares hashes against a pre-defined database (database.txt).
* Automated Quarantine: Automatically moves malicious files to a secure folder to isolate the risk.
* Technical Stack
* Operating System: macOS (MacBook Pro) * Language: Python * IDE: Visual Studio Code * Libraries: hashlib, os, shutil
